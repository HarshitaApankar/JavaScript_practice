# JavaScript Basics

This repository contains simple JavaScript programs that demonstrate how JavaScript interacts with HTML elements. These examples are beginner-friendly and help in understanding the fundamentals of client-side scripting.

## Programs Included

### 1. Change HTML Content using `innerHTML`

* Demonstrates how to change the content of an HTML element.
* Uses the `innerHTML` property to update the text of a paragraph when a button is clicked.

### 2. Show Hidden Content using `display`

* Can use display:none for hiding the content.
* Demonstrates how to show a hidden HTML element.
* Uses the CSS `display` property and JavaScript to make a hidden paragraph visible.

### 3. JavaScript in the `<head>` Section

* Shows how to define a JavaScript function inside the `<head>` tag.
* The function is called using a button click event to modify page content.

### 4. JavaScript in the `<body>` Section

* Demonstrates placing JavaScript inside the `<body>` tag.
* Since the HTML elements are already loaded, the script can directly access and modify them.

## Technologies Used

* HTML5
* JavaScript

## Learning Outcomes

* Understanding basic JavaScript syntax.
* Using `document.getElementById()`.
* Modifying HTML with `innerHTML`.
* Changing CSS properties with JavaScript.
* Handling button click events using `onclick`.
* Understanding the difference between placing JavaScript in the `<head>` and `<body>` sections.

These programs are part of my JavaScript learning journey and serve as a foundation for more advanced web development concepts.
